<center><h1>Challenge-AluraStoreLatam</h1></center>
En este proyecto se realizó el análisis de 4 tiendas de la empresa Alura Store Latam para obtener información relevante que nos permita ayudar a determinar cuál es la tienda con menor rendimiento y poder hacer una recomendación basada en datos al dueño sobre qué tienda debe vender para poder iniciar su nuevo emprendimiento.

## Objetivo.
Identificar la tienda menos rentable a través del análisis de los siguientes factores:
- Ventas Totales
- Evaluaciones de los clientes
- Costos de envío
- Productos más y menos vendidos
- Análisis geográfico de ventas

## Tecnologías usadas
- Python
- Pandas
- Matplotlib
- Folium
- Google Colab

## Desarrollo del proyecto
1. Importación de datos.
   Se importaron los datos desde fuentes externas (archivos CSV) así como las librerías necesarias para el análisis.
2. Análisis exploratorio (EDA)
   Se analizaron los DataFrames para entender su estructura y evaluar los diferentes aspectos que nos ayudarán generar una recomendación.
   - Análisis de facturación:
      Se obtuvieron las ventas totales por tienda y se generó una gráfica que nos dá una comparación visual entre las ventas de cada tienda.
   - Análisis de ventas por categoría:
     Se obtuvo el número de productos vendidos para cada categoría y se usaron representaciones gráficas para mayor claridad.
   - Evaluación de calificaciones de clientes:
     Se obtuvo el promedio de calificación otorgada por los clientes por tienda y se generó una comparación gráfica de las evaluaciones de cada tienda.
   - Productos más y menos vendidos:
     Se obtuvo el nombre y número de productos más y menos vendidos de cada tienda usando una gráfica que nos permite visualizar la información.
   - Costo de envío:
     Se determinó el costo de envío promedio por tienda generando una tabla que permite compararlos de manera fácil y rápida.
   - Análisis de desempeño geográfico:
     Se usaron coordenadas geográficas para generar mapas de calor y poder visualizar las ventas por zona.
3. Visualización de resultados:
   Se utilizaron diferentes tipos de gráficas que nos permiten ver con mayor claridad los diferentes comparativos realizados.
4. Informe final:
   Con base en los análisis anteriores se generó una recomendación para el Sr. Juan, que justifique la tienda que debería vender para iniciar su nuevo emprendimiento.

## Conclusiones
El análisis de la información proporcionó información suficiente para poder generar una recomendacióna al Sr. Juan, siendo esta que con base en los resultados, la Tienda 4 es la que se recomienda vender.
